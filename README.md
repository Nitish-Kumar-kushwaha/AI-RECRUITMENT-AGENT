# AI Recruitment Agent

An AI-powered recruitment assistant built using LangGraph, RAG, ChromaDB, and Streamlit.

This system helps recruiters:
- Search resumes semantically
- Rank candidates intelligently
- Compare applicants
- Generate interview questions
- Explain hiring decisions

---

# Features

- LangGraph AI Agent Workflow
- Resume Semantic Search (RAG)
- Candidate Ranking
- Hiring Report Generation
- Candidate Comparison
- Interview Question Generation
- Explainable AI Recommendations
- Streamlit Chat Interface

---

# Tech Stack

- Python
- LangGraph
- LangChain
- ChromaDB
- Streamlit
- Ollama (Llama3)
- HuggingFace Embeddings

# Installation

## Clone Repository

```bash
git clone <your_repo_link>
```

## Move into Project Folder

```bash
cd ai-recruitment-agent
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Run Ollama

```bash
ollama run llama3
```

---

# Run Streamlit App

```bash
streamlit run app.py
```
